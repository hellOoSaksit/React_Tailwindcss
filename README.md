# 🌐 Web สำหรับการเรียนรู้ - ระบบ Shopping 🛍️✨

เว็บไซต์นี้ถูกสร้างขึ้นเพื่อ **การเรียนรู้** และพัฒนาทักษะการสร้างเว็บแอปพลิเคชันในรูปแบบระบบ **Shopping Online** 🛒 โดยเหมาะสำหรับผู้ที่ต้องการฝึกฝนการพัฒนาระบบจริงผ่านการใช้งาน API ทดลอง

---

## 🚀 ฟีเจอร์เด่น

### 1. ระบบ Shopping 🛍️
- เพิ่มสินค้าลงตะกร้า
- ดูรายละเอียดสินค้า
- ทดลองจัดการระบบการซื้อขาย

### 2. API ทดลอง 🔗
- ใช้ API ตัวอย่างที่เตรียมไว้ให้สำหรับการเรียนรู้ (ไม่มีผลกับข้อมูลจริง)
- สามารถปรับแต่งและทดสอบการเชื่อมต่อ API ได้

---

## 💻 การเริ่มต้นใช้งาน

### 1. Clone โปรเจกต์
```bash
git clone https://github.com/your-username/your-repo.git
```

### 2. ติดตั้ง Dependencies
```bash
npm install
```

### 3. รันโปรเจกต์
```bash
npm start
```

---

## 💡 หมายเหตุ
- API ที่ใช้เป็น **API ทดลอง** ข้อมูลที่ปรากฏในระบบนี้ไม่ได้มีผลกับการซื้อขายจริง
- เหมาะสำหรับการเรียนรู้เท่านั้น
